Users and Groups Notes

## Commands Used
- whoami → shows current user
- groups → shows user groups
- adduser → create new user
- groupadd → create group
- usermod -aG → add user to group
- chmod 770 → group access control

## Concepts Learned
- User = account
- Group = collection of users
- Permissions depend on user + group
